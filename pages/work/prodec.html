Prodec Paints supplies high-end paints for major projects.

I've been working with Prodec Paints for over half a decade, and have successfully completed two website revamps.

Many years ago, when we did the first website revamp, we noticed that there were alot of product changes and copy edits to be made on an ongoing basis. This becomes costly for both the client and developer. The solution was that Prodec would need access to a CMS, which was implemented in the latest revamp. With this "fresh coat of paint" I provided custom WordPress documentation with step-by-step instructions (and pictures) on how and where to make their changes.

I built this website using WordPress with some major styling customizations. The Warranty page required an advanced form that I built custom with PHP and connected it to the Wordpress database.
